# Starting from Surabaya vs Bali

JVTO operates 16 private tour packages — 12 departing from Surabaya, 4 from Bali. The destinations are the same volcanoes and waterfalls. The difference is logistics: where you begin, which direction you travel, and how long the first transfer takes.

---

## Side-by-Side Comparison

| Factor | Surabaya origin | Bali origin |
|---|---|---|
| **Packages available** | 12 | 4 |
| **Duration range** | 1D1N to 6D5N | 3D2N to 5D4N |
| **First transfer** | SUB → Bromo (~3h) or SUB → Bondowoso (~5h) | Bali → Banyuwangi (~5–7h, includes ferry) |
| **Ijen staging hotel** | Riverside Homestay, Bondowoso | Luminor Hotel, Banyuwangi |
| **Ferry crossing** | Only if ending in Bali | Required on Day 1 (Gilimanuk → Ketapang) |
| **Typical route direction** | East (Ijen) → West (Bromo) → return SUB | West (Ijen from Banyuwangi) → West (Bromo) → finish SUB |
| **Cross-island option** | SUB → Bali (`bromo-madakaripura-ijen-3d2n`) | Bali → SUB (all 4 packages end in Surabaya) |
| **3D2N price (2 pax)** | IDR 3,570,000 | IDR 4,050,000 |

---

## When to Start from Surabaya

**Choose Surabaya if:**
- You fly into Juanda Airport (SUB) — most domestic flights from Jakarta, Singapore, and Kuala Lumpur land here
- You want the widest selection — 12 packages vs 4, including short trips (1D1N, 2D1N) not available from Bali
- You plan to return to Surabaya after the tour
- You want to add Malang city or Taman Safari Prigen — these are Surabaya-origin exclusives

**Surabaya advantage**: Shorter first transfer to Bromo (~3 hours). If your package starts with Bromo, you reach the staging hotel by afternoon with a comfortable 09:00 pick-up.

**12 Surabaya packages by duration:**

| Duration | Packages | Destinations covered |
|---|---|---|
| 1D1N | 1 | Bromo only |
| 2D1N | 2 | Bromo+Madakaripura OR Ijen only |
| 3D2N | 3 | Ijen+Bromo+Madakaripura (incl. 1 cross-island to Bali) |
| 4D3N | 3 | Add Papuma+Tumpak Sewu or relaxed pace |
| 5D4N | 2 | Full circuit or Malang extension |
| 6D5N | 1 | Grand tour (all destinations + Malang) |

## When to Start from Bali

**Choose Bali if:**
- You are already in Bali and want to add East Java volcanoes without flying to Surabaya first
- You prefer a one-way overland journey — all 4 Bali packages end in Surabaya (no Bali-to-Bali round trip except `bali/bromo-ijen-3d2n` which returns to Bali)
- Your flight home departs from Surabaya

**Bali trade-off**: The first day is mostly transfer. Bali → Gilimanuk → Ketapang ferry → Banyuwangi takes 5–7 hours depending on traffic and ferry timing. You arrive at the Ijen staging hotel (Luminor Hotel, Banyuwangi) in the evening.

## Price Difference Explained

Bali-origin packages cost more than the equivalent Surabaya-origin package. The difference covers the Gilimanuk–Ketapang ferry crossing and the longer first-day transfer from Bali to East Java. The tour content (destinations, hotels, crew, inclusions) is identical.

**Example — 3D2N Ijen + Bromo + Madakaripura:**
- Surabaya origin: IDR 3,570,000/person (2 pax)
- Bali origin: IDR 4,050,000/person (2 pax)
- Difference: IDR 480,000 — covers ferry + ~2 extra hours of driving

## Cross-Island Routes

If you fly into one city and out of the other, pick the cross-island variant to avoid backtracking:

| Scenario | Package |
|---|---|
| Fly in SUB → fly out Bali | `bromo-madakaripura-ijen-3d2n` (SUB→Bali, 3D2N) |
| Fly in SUB → fly out Bali | `tumpak-sewu-bromo-ijen-4d3n` (SUB→Bali, 4D3N) |
| Fly in Bali → fly out SUB | Any of the 4 Bali-origin packages (all end in Surabaya, except `bromo-ijen-3d2n` which returns to Bali) |
| Fly in SUB → fly out SUB | Most Surabaya packages (round-trip) |
| Fly in Bali → fly out Bali | `bali/bromo-ijen-3d2n` only (3D2N round-trip) |

## Transfer Time Summary

| Segment | Time |
|---|---|
| Surabaya → Bromo (Cemoro Lawang) | ~3 hours |
| Surabaya → Bondowoso (Ijen staging) | ~5 hours |
| Bali → Banyuwangi (Ijen staging, via ferry) | 5–7 hours |
| Ijen area → Bromo area (overland) | ~5 hours |
| Tumpak Sewu → Bromo | 3–4 hours |
| Bromo → Surabaya | ~3 hours |

All transfers in your private vehicle. No shared transport, no public bus connections.
